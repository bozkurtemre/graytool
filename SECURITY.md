# Security Policy

Any security vulnerability discovered in Graytool will be fixed as soon as possible with as minimal impact as possible when reported.

Please ensure you are always running the [latest version available](https://github.com/bozkurtemre/graytool/releases).

Some web browser extension stores may have an older version currently published.

## Security Vulnerability Report

If you have discovered a security vulnerability, please email us at <info@emreb.dev> immediately.
